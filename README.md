# Bank Management System On Console
 A simple, file-based banking application built using Python. This project demonstrates Object-Oriented Programming (OOP), file handling, and menu-driven console interaction.
 Features:-
~ Create New Account
~ User Login
~ Check Balance
~ Add Cash to Account
~ Transfer Money Between Accounts
~ Update Password
~ Update Phone Number
~ Logout System
~ Data Stored in Separate Files (One file per user)

Concepts Used:- 
~ Python Classes & Objects
~ OOP Concepts
~ File Handling (Read, Write, Update)
~ Menu-driven Console Application
~ String Manipulation
~ Error Handling (basic)
          ** How It Works**
1️⃣ Registration
User provides:
Name
Phone Number
Password
System validates inputs and creates a new account file storing user details.

2️⃣ Login
User must enter:
Name
Phone Number
Password
If all details match the file data, the login succeeds.

3️⃣ Add Cash
User enters amount → system updates file and displays new balance.

4️⃣ Transfer Money
User enters:
Amount
Receiver Name
Receiver Phone
System checks receiver file, updates both sender & receiver balances.

5️⃣ Edit Profile
User can:
Change password
Change phone number
The updated information is saved in the same file.
